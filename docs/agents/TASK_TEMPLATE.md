## Checklist

- [ ] Read root `AGENTS.md`.
- [ ] Read `CURRENT_STATE.md` for a new session/goal.
- [ ] Read `GIT_AND_PROVENANCE.md` before Git work.
- [ ] Load only relevant topic docs.
- [ ] Inspect dirty worktree and active session ownership.
- [ ] Inspect current code/tests/reports/ADRs/manifests.
- [ ] Record GitHits/Valyu evidence when required.
- [ ] Define tests and unavailable states.
- [ ] Implement the smallest complete change.
- [ ] Preserve BioMiner artifact-first boundary.
- [ ] Preserve evidence maturity and fingerprints.
- [ ] Run focused tests.
- [ ] Run task gate.
- [ ] Run `git diff --check`.
- [ ] Inspect staged files, rights, network, and provenance.
- [ ] Commit one required subtask.
- [ ] Push at required task boundary.
- [ ] Record remote SHA and limitations.
