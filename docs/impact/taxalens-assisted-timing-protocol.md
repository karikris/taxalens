# TaxaLens-assisted timing protocol

The assisted instrument records the same raw event categories as the manual
baseline while binding each session to an exact TaxaLens Git SHA, campaign, and
task set. Collection is opt-in, local only, and never uploaded by the
instrument.

Record page opens at visible product route changes. Record a duplicate
inspection only when the reviewer opens or evaluates the duplicate context.
Record a field completion when its value is committed, and record a decision
only when the append-only verification event succeeds. Count cache preparation,
source-context expansion, exports, and other deliberate interactions as `other`
actions where no narrower category applies.

Pause when the tab is hidden, the reviewer leaves the task, setup begins, or a
facilitator intervenes. Do not include network waits or automated model work as
human active time unless the participant is actively working during that
interval.

Every exported artifact states whether it came from a human or scripted run.
Scripted runs can measure reproducible interaction burden and software timing;
they cannot establish human productivity. Human sessions remain single-study
observations and do not authorize population savings or scientific-quality
claims.
